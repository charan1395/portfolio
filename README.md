# Sri Charan Rachakonda — Portfolio

Personal portfolio for **Sri Charan Rachakonda**, AI/ML Engineer & Data Scientist.

Live: [sricharanrachakonda.netlify.app](https://sricharanrachakonda.netlify.app/)

## Stack

- React 18 + Vite
- Tailwind CSS
- React Icons
- Deployed on Netlify

## Local development

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # produces /dist
npm run preview  # preview built site
```

## Deploy

Auto-deploys on push to `main` via Netlify Git integration. Build command `npm run build`, publish directory `dist`.
